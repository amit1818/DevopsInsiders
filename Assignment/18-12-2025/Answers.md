1. A website is a collection of several web pages and related content that exist under a common domain name.
   Types of Websites:-
       E-commerce,Social Networking,Entertainment,Blog etc.

2. A 3-tier application is a model that divides an application into three interconnected layers:
    
    Presentation Tier(Frontend): The user interface where the end-user interacts with the system. Here the technologies are used like
HTML,CSS,JavaScript,React,Angular, Next.js,Android / iOS etc.

   
    Logic Tier(Backend): The middle tier of the architecture, also known as the logic tier, handles the application’s core processing, business               rules,       and calculations. Technologies are used here like .NET,Java,Python,Go, Rust etc.
   

    Data Tier(Database): Manages the storage, retrieval, and manipulation of the application’s data, typically utilizing a database. Technologies are used here like MySQL,PostgreSQL,MongoDB,SQL Server etc.
