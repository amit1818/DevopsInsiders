2. A 3-tier application is a model that divides an application into three interconnected layers:
    
    Presentation Tier(Frontend): The user interface where the end-user interacts with the system (e.g., a web browser or a mobile app).
    Logic Tier(Backend): The middle tier of the architecture, also known as the logic tier, handles the application’s core processing, business               rules,       and calculations.
    Data Tier(Database): Manages the storage, retrieval, and manipulation of the application’s data, typically utilizing a database.
